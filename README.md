# 네이버 카페 대문 — 주간 스케줄 생성기 (v2)

구글시트의 주간 스케줄 → 네이버 카페 대문에 붙여넣을 **칸별 클릭 가능한 HTML 표**.
**v2 원칙: "상태를 시트로, 도구를 URL로"** — 운영진 누구나 열면 같은 결과. 기획: [기획서-v2.md](기획서-v2.md)

## 매주 루틴 (이게 전부)

1. `카페대문.html` 열기 (또는 GitHub Pages URL) — 열리면 시트에서 자동 로드
2. 주 탭 선택 → 오타 있으면 카드 클릭해서 수정
3. **HTML 복사** → 카페 관리 → 대문 → HTML 편집 → 붙여넣기 → 저장

## 시트 구조 (운영진 전원 공유 = 유일한 진실)

```
0615 ~ 0621, 0622 ~ 0628, …   주차 탭 (기존 그대로. 멤버당 시간행+제목행)
members                        멤버 | 배경색 | 글자색 | 채널URL | 이미지URL
config (선택)                  [헤더|주간 스케줄표] [배지|…] [로고|https…]
```

- **members 탭 만들기**: 도구의 설정⚙ → "members 탭용 표 복사" → 시트에 `members` 탭 만들고 A1에 붙여넣기.
  이후 색/URL/이미지는 시트에서 수정 → 운영진 전원에게 반영.
- 새 주차 탭이 생기면: 주 탭 바의 "새 주 탭 URL/gid"에 그 탭 URL 붙여넣고 ＋.
- 도구 안에서의 수정(카드 클릭, 설정 변경)은 **이 브라우저에만 임시** — 원본은 언제나 시트.

## 파일 구조

```
src/generate.mjs   순수 생성기 (의존성 0). 네이버 생존 마크업만 출력. SURVIVE 플래그.
src/sheet.js       시트 연동: 주차 탭 + members/config 탭 (gviz JSONP, file:// OK)
src/csv.mjs        CSV 파서 (의존성 0)
src/ui.js          v2 UI: 무상태 부팅(시트 자동 동기화) + 주 탭 + 미리보기 클릭 편집 + 경고 배지
build.mjs          node build.mjs → 카페대문.html + index.html (Pages용) 인라인 빌드
cli.mjs            디버그용 CLI (글자 자동축소 없음 — 정식 경로는 웹)
spike.html/2       네이버 생존 실측 프로브 (완료, 재확인용)
```

## 네이버 생존 규칙 (전부 실측 — 이 도구의 모든 출력이 이 규칙 안에 있음)

| 마크업 | 결과 |
|---|---|
| `<a>` 텍스트 링크(color/밑줄만) · 배경색 · 모서리 · 그림자 · 외부 img | ✅ 생존 |
| `data:` 이미지 · **`<a>`에 background/display(=카드 전체 클릭)** | ❌ 삭제 |

카드 규칙(확정): 60px 고정 직사각형 · 이름 좌상단 · 시간 우상단 · 내용이 넘치면 카드가 아니라 **글자가 줄어듦**(복사 시점에 px로 구워넣음).

## 운영진 공유 (GitHub Pages)

```bash
node build.mjs && git add -A && git commit -m update && git push
# repo Settings → Pages → main 브랜치 루트 → 생성된 URL을 운영진에게 공유
```
