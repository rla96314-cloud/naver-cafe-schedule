네이버 대문 생존 스파이크 — Phase 0

설계문서가 못박은 단 하나의 게이트. 아래 HTML을 본인 카페 대문에 붙여넣고 저장→새로고침 후, 무엇이 살아남는지 눈으로 확인합니다. 이 결과가 생성기 출력 포맷을 확정합니다.

① 이 HTML을 복사

② 카페에 붙여넣고 확인

  1. 카페 관리 → 대문 → HTML 편집 모드
  2. 위 HTML 붙여넣기 → 저장 → 페이지 새로고침(에디터가 sanitize한 결과를 봐야 함)
  3. 각 번호 칸이 아래 "기대 모습"대로 남았는지 / 깨졌는지 체크

③ 무엇을 보는가 (붙여넣은 결과와 대조)

#테스트PASS = 이렇게 보이면 살아남음
①일반 <a href> 링크밑줄 링크가 보이고 클릭하면 이동
②셀 배경색칸이 주황색으로 칠해짐
③border-radius칸 모서리가 둥글게
④box-shadow칸 아래 그림자
⑤외부 <img src=https>외부 이미지가 표시됨
⑥data: base64 이미지파란 사각 이미지가 표시됨
⑦<a>로 감싼 컬러 카드색칠된 카드 전체가 클릭되어 이동
