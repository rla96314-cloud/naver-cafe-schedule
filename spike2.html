스파이크 2차 — "카드 전체 클릭" 살릴 수 있나

1차에서 ⑦(블록 <a> 카드)이 죽었지만 변수가 4개나 섞여 있었음 (display:block + 배경 + 그림자 + <div> 자식). 어느 게 범인인지 좁혀, 카드 전체를 누르게 만들 수 있는지 확정한다. 1차처럼 붙여넣고 A~E 중 뭐가 살아남는지 보고.

② 무엇을 보는가

#구조PASS = 이렇게 되면 살아남음
A<a display:block> + 배경, 자식은 텍스트만(div 없음)색칠 카드 전체 클릭
B<a display:inline-block; width:100%> + 배경, 텍스트만색칠 카드 전체 클릭
C<a>(display 기본) + 배경, 텍스트만색칠된 링크 영역 클릭
D<div 배경><a>텍스트</a></div> (현재 채택 구조)카드 보이고 텍스트 클릭 (대조군, PASS 기대)
ED + text-decoration:underline 강제링크에 밑줄 보임(모바일 가시성)

결과: A PASS / B FAIL … 형태로. A·B 중 하나라도 PASS면 카드 전체 클릭을 부활시키고, 다 FAIL이면 현재 D 구조로 확정한다.
